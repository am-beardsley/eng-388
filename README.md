# eng-388
ENG 388: Writing for the Web is an upper-level undergraduate course at Western Illinois University.
Our course explores the questions including but not limited to: 
What's your favoriate food? 
Where does it originate? 
What is the most popular food on campus? 

Dr. Beardsley maintains the project, and students in the course contribute.
